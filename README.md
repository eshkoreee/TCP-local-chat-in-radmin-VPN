# TCP-local-chat-in-radmin-VPN
я в форме обучения для закрепления материала написал собственный консольный TCP чат. в нем можно ввести айпи хоста свое имя. поддерживает кириллицу. больше подробностей найдете на README

для запуска программы потребуется следуещие программы: radmin VPN и инретпретатор python.
один компьютер хост создает локальную сеть в radmin VPN и участники к ней подключаются. как сделать это можете найти на ютубе
далее как скачать интерпретатор пайтон тоже можете найти на ютубе не вижу смысла бъяснять это здесь
далее когда все участники подключились хост запускает файл TCPchat_server а затем все участники в том числе хост запускают файл TCPchat_client
в серверной части не нужно ничего записывать
в клиентской консоли нажно записать айпи хоста и нажать enter. затем напишите свой никнейм и после этого вы подключитесь к локальному чату
тому кто хостит в клиенте нужно прописать 127.0.0.1 как локалхост
там можете спокойно переписыватся а если кто то захочет покинуть чат нужно прописать команду quit
на консоли сервера выскочит исключение но это нормально оно оповещает что один из клиентов оборвал соединение
пользуйтесь на здоровье
